# vue3-music

分支介绍：

1. `main` 主分支 包含除 ts 外的所有代码
2. `plugins-vue3` 使用 vue3 构建插件
3. `Vue3.0` 使用 Vue3 改造
4. `Vue2.0` 使用 Vue2 编写
5. `ts-Vue3` 使用 ts 改造 Vue3 代码

## Project setup

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn serve
```

### Compiles and minifies for production

```
yarn build
```
